## Описание

«Игры разума» — это набор из пяти консольных мини-игр. Каждая игра запрашивает у пользователя ответы на математические вопросы. Нужно дать правильные ответы три раза подряд.

## Минимальные требования

- PHP 8.1 или выше
- Composer

Установите зависимости:
make install

BRAIN EVEN

Пользователю показывается случайное число. Нужно ответить yes, если число чётное, или no — если нечётное.
Запуск: make brain-even

[![asciicast](https://asciinema.org/a/RybaG538laTLByC2IsmlWtzPW.svg)](https://asciinema.org/a/RybaG538laTLByC2IsmlWtzPW)

BRAIN CALCULATOR

Пользователю показывается случайное математическое выражение, например 35 + 16. Нужно вычислить и записать правильный ответ.
Запуск: make brain-calc

[![asciicast](https://asciinema.org/a/RPP3Myr8pSeMX8DjpKtMzeR01.svg)](https://asciinema.org/a/RPP3Myr8pSeMX8DjpKtMzeR01)

BRAIN GCD

Пользователю показывается два случайных числа, например 25 50. Нужно вычислить и ввести наибольший общий делитель этих чисел.
Запуск: make brain-gcd

[![asciicast](https://asciinema.org/a/L69muU2rivKfN1QqYeuj5jwpj.svg)](https://asciinema.org/a/L69muU2rivKfN1QqYeuj5jwpj)

BRAIN PROGRESSION

Пользователю показывается ряд чисел, образующий арифметическую прогрессию, где одно число скрыто двумя точками. Нужно определить это число.
Запуск: make brain-progression

[![asciicast](https://asciinema.org/a/qCCP1KCaQdGRO8guUutcGIk2V.svg)](https://asciinema.org/a/qCCP1KCaQdGRO8guUutcGIk2V)

BRAIN PRIME

Пользователю показывается число. Нужно ответить yes, если число простое, или no — если составное.
make brain-prime
